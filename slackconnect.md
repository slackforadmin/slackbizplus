Slack Connect 설정
Slack connect 에 대한 조직 레벨의 설정을 할 수 있습니다. 세부항목별로 설명합니다.

범용
멤버 권한
외부 조직에서 만든 워크플로 사용
기본값 : 활성화됨 (외부 조직에서 만든 워크플로를 사용할 수 있도록 허용 합니다.)
선택 가능한 옵션 : 타사 커넥터 단계가 포함된 워크플로 (3rd 커넥터가 포함된 워크플로를 실행할 수 있는지 여부 설정)

외부 사람들 권한
<Grid Name> 에서 만든 워크플로 사용
기본값 : 활성화됨 (외부 조직의 인원이 내 조직의 워크플로를 사용할 수 있도록 허용합니다.)
선택 가능한 옵션 : 타사 커넥터 단계가 포함된 워크플로 (외부 조직의 인원이 3rd 커넥터가 포함된 내 조직의 워크플로를 실행할 수 있는지 여부 설정)

설정
캔버스
기본값 : 활성화 됨
선택 가능한 옵션 : 비활성화
