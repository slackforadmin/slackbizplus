
# 프로필 구성

Slack 아틀라스에서는 세 가지 방법 중에서 선택하여 프로필 필드를 업데이트할 수 있습니다.

SCIM API
IDP에서 구성된 프로필 필드 또는 사용자 지정 SCIM 커넥터를 통해 자동으로 채웁니다.
Web API
Web API를 사용해 프로필 필드를 설정 또는 검색합니다.
멤버 편집 
멤버는 수동으로 프로필 필드를 업데이트할 수 있습니다.
