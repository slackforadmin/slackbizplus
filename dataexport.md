
# 데이터 가져오기 및 내보내기
Slack 워크스페이스의 데이터 가져오기/내보내기 관리 화면입니다.

<img width="1356" alt="image" src="https://github.com/user-attachments/assets/b7f2e99c-e304-4ad8-a9d5-9eec5b570342" />

1. 가져오기 탭다른 서비스 또는 Slack 워크스페이스의 데이터를 MinaBiz+ 워크스페이스로 가져오는 기능입니다.<br>
가져오기 옵션:<br>
Slack: 다른 Slack 워크스페이스에서 메시지 내역과 파일을 가져올 수 있어요.<br>
CSV/텍스트 파일: 데이터베이스 등에서 사용자 지정 형식의 메시지를 가져올 수 있어요.<br>
각 옵션마다 "가져오기" 버튼이 있으며, 가져오기 전에 메시지, 파일, 계정을 선택할 수 있습니다.<br>

2. 내보내기 탭워크스페이스에서 메시지 및 파일을 내보내는 기능입니다.<br>
중요 공지:<br>
평가판 기간에는 비공개 채널 및 대화 내보내기가 일시적으로 제한됩니다.<br>
보안상의 이유로 데이터 내보내기가 제한되며, 워크스페이스를 플랜 Business+ 플랜으로 전환하면 전체 액세스 권한이 부여됩니다.<br>
Slack 지원 팀에 문의하여 도움을 받을 수 있어요.<br>
포함되는 항목:<br>
공개 채널에 전송된 메시지 및 파일 링크<br>
앱 활동<br>
포함되지 않는 항목:<br>
비공개 채널에 전송된 메시지 및 파일 링크<br>
다이렉트 메시지에 전송된 메시지 및 파일 링크<br>
편집 및 삭제 로그<br>
내보내기 방법:<br>
스탠다드 내보내기를 사용하여 워크스페이스에서 메시지 및 파일을 내보낼 수 있어요.<br>
채널 메시지 형식에 대한 자세한 내용은 API 문서를 참조하세요.<br>
내보내기 날짜 범위를 선택할 수 있는 드롭다운 ("하나 선택...")과 "내보내기 시작" 버튼이 있습니다.<br>
