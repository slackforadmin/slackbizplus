
# 데이터 가져오기 및 내보내기
Slack 워크스페이스의 데이터 가져오기/내보내기 관리 화면입니다.

<img width="1356" alt="image" src="https://github.com/user-attachments/assets/b7f2e99c-e304-4ad8-a9d5-9eec5b570342" />

1. 가져오기<br>
 - 다른 서비스 또는 Slack 워크스페이스의 데이터를 해당 워크스페이스로 가져오는 기능입니다.<br>
[가져오기 옵션]<br>
Slack : 다른 Slack 워크스페이스에서 메시지 내역과 파일을 가져올 수 있습니다.<br>
[한 Slack 워크스페이스에서 다른 워크스페이스로 데이터 가져오기](https://slack.com/intl/ko-kr/help/articles/217872578-%ED%95%9C-Slack-%EC%9B%8C%ED%81%AC%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4%EC%97%90%EC%84%9C-%EB%8B%A4%EB%A5%B8-%EC%9B%8C%ED%81%AC%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%A1%9C-%EB%8D%B0%EC%9D%B4%ED%84%B0-%EA%B0%80%EC%A0%B8%EC%98%A4%EA%B8%B0) <br>
CSV/텍스트 파일 : 데이터베이스 등에서 사용자 지정 형식의 메시지를 가져올 수 있습니다.<br>

2. 내보내기<br>
 - 워크스페이스에서 메시지 및 파일을 내보내는 기능입니다.<br>
[내보내기 방법]<br>
스탠다드 내보내기를 사용하여 워크스페이스에서 메시지 및 파일을 내보낼 수 있습니다.<br>
채널 메시지 형식에 대한 자세한 내용은 API 문서를 참조하세요.<br>
