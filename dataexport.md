 # 데이터 내보내기 요청
워크스페이스에 사용할 수 있는 내보내기 옵션은 다음과 같습니다.

 - 공개 채널
워크스페이스 소유자와 관리자는 공개 채널의 메시지와 파일 링크를 JSON 형식으로 내보낼 수 있습니다.

 - 모든 채널과 대화
법률과 규정을 준수하기 위해 워크스페이스 소유자는 JSON 형식으로 공개 채널, 비공개 채널, DM의 메시지와 파일 링크에 사용할 수 있는 셀프 서비스 데이터 내보내기 도구에 대한 액세스를 신청할 수 있습니다. 이 내보내기 유형이 승인되면 워크스페이스 소유자는 반복적인 데이터 내보내기를 예약할 수도 있습니다.

 - 채널 감사 보고서
워크스페이스 소유자는 이전 계획 또는 기타 워크스페이스 관리 요구 사항에 대한 채널의 CSV 리스트와 해당 세부정보에 대하여 내보내기를 요청할 수 있습니다.
